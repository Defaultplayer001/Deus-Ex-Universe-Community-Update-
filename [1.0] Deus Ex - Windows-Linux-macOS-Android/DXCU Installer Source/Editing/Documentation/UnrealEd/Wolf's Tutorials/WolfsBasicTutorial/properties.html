Properties Window


This is the a generic Properties Window in UnrealEd. You can open this dialog for ALL actors. Though each Properties window looks the same there are specific fields that pertain only to that actor and those fields are unique to that actor's Properties Windows.



This is the Properties Window for one of Barrels in out level. Note the similarities with the generic window above, but note the differences. I have highlighted the actor-specific fields, which are the ones you should be concerned with for this actor.

Here I have highlighted the actor-specific fields for a light actor.
Note: Light Color and Lighting appear in all the Properties windows, but most actors will not emit light.
