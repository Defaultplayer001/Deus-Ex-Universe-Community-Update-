Abstürze beim Speichern/Laden

Gelegentlich werden Kartendateien bei der Installation beschädigt. Wir nehmen an, dass dies durch bestimmte Festplatten-/Festplatten-Controller/CD-ROM-Kombinationen ausgelöst wird. In diesem Fall stürzt das Spiel beim Laden einer neuen Karte ab. Im Folgenden finden Sie einige Tipps zur Lösung des Problems. Befolgen Sie diese Hinweise in der angegebenen Reihenfolge. Wenn die erste Lösung nicht funktioniert, probieren Sie die nächste aus.

 

Verringern Sie beim Laden neuer Karten die Detailstufe der Texturen

 

Gelegentlich lässt sich das Problem durch die Verringerung der Welt- und Detailtexturen beim Übergang in eine andere Karte beheben. Nach dem Laden der neuen Karte können Sie die normale Detailstufe wiederherstellen.

 

 

(Nur für erfahrene Anwender!) Installieren Sie das Spiel von der Festplatte

 

Wenn weiterhin Probleme auftreten, oder Sie nichtssagende Fehlermeldungen erhalten, sollten Sie versuchen, Deus Ex nicht von der CD-ROM, sondern von Festplatte zu installieren. Dazu gehen Sie folgendermaßen vor:

 

1) Vergewissern Sie sich, dass Sie über ausreichend Festplattenspeicher für die Installation und den gesamten Inhalt der CD-ROM verfügen (also etwa 1,4 GB).

2) Legen Sie einen neuen Ordner namens "Deus Ex Install" an.

3) Kopieren Sie den Inhalt der CD-ROM in diesen Ordner.

4)Führen Sie anschließend die Setup-Datei im Ordner "Deus Ex Installation" aus und installieren Sie das Spiel.

5)Löschen Sie nach der Installation den Ordner "Deus Ex Installation".

 

 

(Nur für erfahrene Anwender!) Kopieren Sie die Karte von der CD-ROM

 

1) Stellen Sie anhand der Fehlermeldung fest, welche Karte den Absturz verursacht. Ein Teil der Fehlermeldung enthält vermutlich eine Zeichenfolge, die in etwa so aussehen könnte: 01_NYC_UNATCOIsland. Suchen Sie nach einem Eintrag, der mit zwei Ziffern beginnt und nach einer Ortsangabe klingt. Wenn Sie dann ".dx" zu diesem Namen hinzufügen, erhalten Sie den Dateinamen der Karte. In diesem Fall ist der Dateiname der Karte "01_NYC_UNATCOIsland.dx".

2) Kopieren Sie die entsprechende Karte aus dem Ordner "Maps" der CD-ROM in den gleichnamigen Ordner im Installationsverzeichnis des Spiels. Wenn Ihr CD-ROM-Laufwerk den Buchstaben "F" trägt und Sie das Spiel im Verzeichnis "C:\DeusEx" installiert haben, kopieren Sie die Karte von "F:\Maps\01_NYC_UNATCOIsland.dx" nach "C:\DeusEx\Maps\01_NYC_UNATCOIsland.dx".

 

3) Gespeicherte Spiele, die Daten der fehlerhaften Karte enthalten, könnten ebenfalls beschädigt sein. Löschen Sie daher alle Spielstände, die auf die entsprechende Karte zugreifen (Sie erkennen diese an der Dateierweiterung ".dxs"). Löschen Sie zunächst den aktuellsten Spielstand (das Spiel mit der höchsten Nummer) und wiederholen Sie den Vorgang, bis Sie auf einen Spielstand ohne die fragliche Karte stoßen

 

Wenn in Ihrem Ordner "Save" beispielsweise 20 gespeicherte Spielstände enthalten sind, beginnen Sie mit Save 0020 und arbeiten Sie sich gegebenenfalls bis zu Save0001 vor. Wenn die Karte "01_NYC_UNATCOIsland.dxs" in Save0020 und Save0019 enthalten ist, nicht aber in Save0018, müssen Sie nur die Ordner Save0020 und Save0019 löschen. Es ist durchaus möglich, dass die fehlerhafte Karte in keinem gespeicherten Spiel verwendet wird. In diesem Fall entfällt das Löschen von Spielständen.

 

4) Laden Sie anschließend den aktuellsten Spielstand und setzen Sie das Spiel fort.

 

 

 

 

Empfehlungen für D3D-Probleme

 

Sollte Deus Ex auf Ihrem System sehr langsam laufen, versuchen Sie, die Leistung des Spiels mit folgenden Einstellungen zu verbessern:

 

Klicken Sie im Hauptmenü auf Einstellungen, anschließend auf "Grafik":

 

 

Bildschirmauflösung - Wählen Sie hier 640x480 oder 800x600.

Farbtiefe der Texturen - Wählen Sie 16-bit.

Welttexturen - Setzen Sie diesen Wert auf "Mittel".

Detailtexturen - Schalten Sie diese Option AUS

 

 

Wenn Sie mit der Direct3D-Leistung des Spiels nicht zufrieden sind, können Sie versuchen, OpenGL als Rendering-Gerät zu verwenden. OpenGL wird zwar von unserem Kundendienst nicht unterstützt, kann aber unter bestimmten Umständen vor allem bei Grafikkarten mit Nvidia-Chipsatz (z. B. GeForce) eine deutliche Leistungssteigerung bewirken.

 

Wenn Sie eine 3DFx-Grafikkarte besitzen, ist OpenGL NICHT ZU EMPFEHLEN. Wählen Sie stattdessen "Glide", um die Leistung des Spiels zu optimieren.

 

Um OpenGL über das Hauptmenü von Deus Ex zu aktivieren, wählen Sie im Einstellungen-Bildschirm "Grafik" und anschließend "Rendering-Gerät". Deus Ex wird nun automatisch neu gestartet. Anschließend erscheint der Bildschirm "Grafikkonfiguration". Wählen Sie hier die Option "Alle Geräte zeigen". Markieren Sie in der Liste "OpenGL-Unterstützung", bestätigen Sie Ihre Wahl zweimal mit "Weiter" und schließen Sie den Vorgang durch Anklicken der Schaltfläche "Fertig" ab. Deus Ex versucht dann, beim Start den OpenGL-Modus zu initialisieren.

 

Ist dies nicht möglich, erscheint eine Fehlermeldung oder das Spiel wird im Fenstermodus gestartet und läuft außerordentlich langsam. Dies bedeutet, dass OpenGL von Ihrer Grafikkarte oder den installierten Grafiktreibern nicht unterstützt wird. Wählen Sie in diesem Fall als Rendering-Gerät wieder Direct3D. Starten Sie Deus Ex dazu im abgesicherten Modus und wählen Sie die Option "Direct3D".

 

(Nur für erfahrene Anwender!) Wenn Sie den OpenGL-Modus mit einer TNT- oder Radeon-Karte ausführen, können Sie folgendes versuchen, um die Leistung zu verbessern:

 

 

Speichern Sie eine Kopie der DeusEx.ini-Datei ab, die sich im Verzeichnis "System" des Installationsverzeichnisses befindet. Kommt es bei den Anpassungen zu Problemen, können Sie diese Kopie dazu verwenden, die Einstellungen wiederherzustellen.

 

Sie können die DeusEx.ini-Datei mit einem Texteditor bearbeiten.

 

Suchen Sie nach einem Textblock, der mit folgender Zeile beginnt: "[OpenGLDrv.OpenGLRenderDevice]".

 

Ein paar Zeilen darunter, im selben Textblock, finden Sie folgende Zeile: "UseTNT=0". Ändern Sie dies zu "UseTNT=1".

 

 

(Hinweis: Wenn Deus Ex im OpenGL-Modus läuft, wird beim Laden des Spiels kein Screenshot des gespeicherten Spiel angezeigt. Das ist normal.)
