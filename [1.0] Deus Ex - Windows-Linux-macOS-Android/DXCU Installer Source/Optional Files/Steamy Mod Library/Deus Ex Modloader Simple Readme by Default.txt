1. A standard mod.int file, used to add mods to the mod loader list, is as follows:
[Public]
Object=(Name=Redsun 2020,Class=Class,MetaClass=DeusExML.MLGameInfo,Description=Map=MLRedsun2020.dx;)

2. Edit the Name and Map fields as desired. You can also add Splash= to set a custom Dx.Dx file. (The Logo map file)

3. Name as desired and distribute with mod, making sure it's placed in the root system folder alongside the modloader
