Deus Ex パッチ　Readme 
2001年 3月
Version 1.0

本パッチは、Deus Ex日本語版にマルチプレイモードを追加し、いくつかの問題を修正、変更しています。
なお、このパッチは日本語版のみをサポートするものであり、他の言語のバージョンでは正常に起動しない場合があるので、あらかじめご了承ください。

インストール方法
本パッチは、自動解凍形式です。クリックすると、セットアップ画面に切り替わり、画面に指示に従い、インストールを開始します。

マルチプレイに関して
本パッチをインストール後に、ゲームを起動すると、メインメニューに[Multiplayer]という項目が追加されます。
[Multiplayer]をクリックすると、マルチプレイ用のメニュー画面が開きます。

本パッチをインストールすることで、インターネット、LANの各モードでマルチプレイをお楽しみ頂くことができます。
以下の手順でゲームをお楽しみください。

A.キャラクター作成
ゲームを始める前に[Multiplayer Main Menu]から[Player Setup]を選択します。
1)アルファベットで名前を入力します。
2)お使いのPCに接続されている回線の種類を選択してください。モデム接続か、それ以外かを選択します。
3)[Help Message]を[On]にすると、ゲーム中に状況などをテキストメッセージで知らせてくれます。
4)マルチプレイでは、個人で戦うデスマッチのプレイヤーのキャラクターを選択できます。[Non-Team Model]で、通常デスマッチの際のキャラクターを選択します。キャラクターによる能力の差はありません。
5)チームデスマッチを行う場合のプレイヤーのキャラクターを設定することができます。[Team Model]でキャラクターを選択します。NSFチームかUNATCOチームを選択します。ゲームに参加した場合、選択したチームに必ず所属します。AUTOに設定しておくと、ゲーム終了後にランダムでチームが代わります。

また[Multiplayer Main Menu]から[Aug Setup]を選択することで、マルチプレイ中に使用できるAugを設定できます。
ただし、ホストプレイヤーによってAugの数が制限される場合がるので、その場合は、セットアップ画面の[Augmentation Priorities]の上から順番に、ホスト側で設定したAugが使えるようになります。


B.キーボード設定
マルチプレイの機能がゲームに追加されることで、キーボードの設定に以下の新たな新たな項目が設定されます。
メインメニューから[Setting]→[Keyboard/Mouse]を選択してください。

・マルチプレイスコア確認
・チーム内メッセージ
・メッセージ
・スキルメニュー
・キル数確認

C.ゲームを検索
上記、A、Bの設定を行うと、マルチプレイゲームを始める準備ができます。Deus ExをプレイするPCが、LANもしくはインターネットに接続されていることを確認してください。
メインメニューから[Multiplayer]を選択し、[Internet Game]、[LAN Game]のいづれかを選択してください。
いづれの場合も、ゲーム検索画面が現れます。
ウインドウには、現在、InternetまたはLAN上に作成されたホストゲームが表示されます。画面に表示されたホストゲームに参加するには[Join Game]をクリックしてください。
[Host Game]をクリックすると、ホストゲームを作成することができます。
*インターネットゲームでは、GamespyでマッチングされるDeus Exマルチプレイゲームを検索します。画面上で検索され、接続をお試し頂く事ができますが、お使いのゲームのバージョンがホスト側と異なる場合、接続できない場合があります。ご注意ください。

D.ホストゲーム
ゲーム検索画面から[Host game] を選択すると、ゲームの設定画面になります。ホストプレイヤーとなり、ゲームのルールを設定してください。

　

パッチ 修正点
本パッチでは、以下の内容が修正、変更されています:

  • F1、 F2 キーは使用できなくなりました。
  • D3DDrv.dll パッチが組み込まれました。
  • 新しい OpenGL コードが組み込まれました。
  • Win95aでゲームが起動できるようになりました。

　

セーブ/ロードに関して
インストールの際、 マップファイルに問題が発生する場合があります。これは、ハードディスク、ハードディスクコントローラ、CD-ROMの組み合わせによって引き起こされている可能性があります。
問題が発生した場合、以下の方法で問題が修正される場合があります。
  • テクスチャーレベルを低くします

    ゲームをスタートする前に、メインメニューから[Setting]→[Display]の順番に選択していきます。[Display Settings]のメニューから、[World Texture Detail],[Object Texture Detail]を[Low]に設定し、[Detail Textures]を[Off]にしてください。

　

D3Dに関して
  • お使いのマシンでDeus Exの起動が遅いと感じられる場合、以下の方法をお試しいただくことで、問題が解決される場合があります。

    ゲームをスタートする前に、メインメニューから[Setting]→[Display]の順番に選択していきます。[Display Settings]のメニューから、以下のように設定を変更します。
    • Screen Resolution - 640x480 もしくは 800x600に設定します。.
    • Texture Color Depth -16ビットに設定します。
    • World Texture Detail - Mediumに設定します。
    • Detail Textures - オフに設定します。

  • Direct3Dのパフォーマンスに問題がある場合、OpenGLを使用することで、問題が解決できる場合があります。けれども、Deus ExではOpenGLをサポートしてはいませんが、Nvidea、GeForceなどチップセットを使用している場合、パフォーマンスが向上することが確認されています。 
    OpenGL は、3ｄｆｘのチップセットを使用したビデオカードでの動作は保証いたしかねます。3dfxのビデオカードを使用している場合はGlideを使用することをお勧め致します。

    OpenGLドライバを使用可能にするには、メインメニューから[Setting]→[Display]の順番に選択していき、[Rendering Device]を選択します。自動的に画面がウインドウ画面に切り替わり、使用できるビデオデバイスを表示します。OpenGLドライバを選択して、画面の指示に従い設定します。 

    もし、上記の設定を行いエラーが出る場合は、お使いのビデオカード、あるいはドライバがOpenGLをサポートしていない可能性があります。もし、OpenGLをサポートしていなければ、D3Dを使用する必要があります。
    Deus Exをセーフモードで起動し、D3Dドライバを設定しなおす必要があります。
