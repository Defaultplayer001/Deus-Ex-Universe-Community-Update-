Recommandations relatives aux plantages lors de la sauvegarde/du chargement

Il est possible que, au cours de l'installation, certains fichiers de zones aient été corrompus. Nous pensons que cela peut être causé par certaines combinaisons de disque dur, contrôleur de disque dur et lecteur de CD-ROM. Dans ce cas, le jeu plante ou se bloque lors des changements de zone. Quelques suggestions pour résoudre ce problème suivent ci-dessous. Essayez-les dans cet ordre ; si la première méthode ne marche pas, essayez la seconde, et ainsi de suite.

 

Réduction des détails des textures pendant la transition

 

Nous avons entendu parler de succès partiels de passage de zones sans plantage, en réduisant au minimum les détails Décors et Modèle. Ces niveaux de détails peuvent être ensuite ramenés à leur état normal, après la transition.

 

 

(Seulement pour les utilisateurs confirmés !) Installation à partir du disque dur

 

En cas de problèmes récurrents ou sans messages d'erreur spécifiques, nous avons entendu parler de succès partiels avec une installation à partir d'un disque dur, par opposition à l'installation normale à partir du CD-ROM de Deus Ex. Il suffit de suivre les instructions suivantes :

 

1) Assurez-vous que vous disposez de suffisamment d'espace libre sur votre disque dur pour le jeu installé et l'image du disque, soit environ 1,4 Go.

2) Créez un répertoire appelé " Deus Ex Install " qui recevra l'image du disque.

3) Copiez le contenu du CD dans ce répertoire.

4) Exécutez ensuite le fichier d'installation présent dans " Deus Ex Install " comme vous le feriez à partir du CD.

5) Effacez le répertoire " Deus Ex Install " après la fin de l'installation.

 

 

(Seulement pour les utilisateurs confirmés !) Recopier la zone depuis le CD

 

1) Trouvez la zone incriminée. Elle est indiquée dans le message d'erreur. Vous devriez y voir quelque chose ressemblant à " 01_NYC_UNATCOIsland ",  à savoir une chaîne de caractères commençant par deux chiffres et contenant le nom d'une zone (en anglais). En ajoutant " .dx " à la fin, vous obtenez le nom du fichier recherché. Dans notre exemple, il s'agit de " 01_NYC_UNATCOIsland.dx ".

2) Copiez cette zone depuis le répertoire " Maps " installé sur votre disque dur. Dans notre exemple, si votre CD-ROM est " F: " et que vous avez installé le jeu dans " C:\DeusEx ", vous devriez copier le fichier " F:\Maps\01_NYC_UNATCOIsland.dx " vers " C:\DeusEx\Maps\01_NYC_UNATCOIsland.dx ".

 

3) Toutes les sauvegardes contenant des données issues de la zone corrompue risquant de l'être également, il est donc conseillé d'effacer tout répertoire de sauvegarde contenant le fichier de zone (avec une extension de type " .dxs " au lieu de " .dx "). Commencez par vos sauvegardes les plus récentes, c'est-à-dire celles dont le numéro est le plus élevé, et progressez en arrière dans le temps, jusqu'à ce que vous rencontriez une sauvegarde ne contenant pas le fichier corrompu.

 

Par exemple, si votre répertoire " Save " contient 20 sauvegardes, commencez avec le répertoire Save0020 et remontez jusqu'à Save0001. Si Save0020 et Save0019 contiennent le fichier " 01_NYC_UNATCOIsland.dxs ", mais pas Save0018, supprimez les répertoires Save0020 et Save0019. Il est tout à fait possible qu'aucun répertoire de sauvegarde ne contienne le fichier en question ; dans ce cas, vous n'aurez à en effacer aucun.

 

4) A partir de là, vous devriez pouvoir charger vos sauvegardes les plus récentes et continuer à jouer sans problèmes.

 

 

 

 

Recommandations relatives aux ralentissements dus à la 3D

 

En cas de baisse de performance en cours de jeu, il est possible d'altérer quelques réglages pour essayer de remédier à ce problème.

 

Dans le Menu Principal, cliquez sur Configuration, puis Graphismes :

 

 

Résolution de l'écran : sélectionnez une résolution de 640x480 ou de 800x600.

Profondeur de la texture des couleurs : sélectionnez 16 bits.

Détails de la texture des décors : sélectionnez Moyen(ne).

Textures détail : désactivez cette option.

 

 

Si les performances de Direct3D ne vous semblent pas optimales, sélectionnez OpenGL comme protocole 3D. Ce dernier n'est pas officiellement supporté par Deus Ex, mais peut apporter des gains de performances significatifs dans certaines circonstances (particulièrement avec des chipsets Nvidia, comme la GeForce).

 

Il N'EST PAS RECOMMANDE aux possesseurs de cartes graphiques à base de 3DFx d'utiliser OpenGL. Ces derniers devraient utiliser Glide, pour des performances optimales.

 

Pour activer OpenGL, accédez au Menu Principal de Deux Ex, Configuration puis Affichage, et enfin Périphériques de rendu. La boîte de dialogue de sélection de carte graphique s'affichera aussitôt. Sélectionnez l'option Afficher tous les périphériques. Puis sélectionnez mode OpenGL, puis cliquez sur Suivant à deux reprises et enfin sur Terminer. Deus Ex essayera d'utiliser OpenGL.

 

S'il n'y arrive pas (message d'erreur ou affichage extrêmement lent dans une fenêtre), cela signifie que votre carte graphique ou son pilote n'accepte pas OpenGL. Dans ce cas, il faudra resélectionner Direct3D. Pour ce faire, quittez puis lancez Deux Ex à partir du raccourci du Mode Sécurisé, et sélectionnez Direct3D.

 

(Seulement pour les utilisateurs confirmés !) Si vous utilisez le mode OpenGL avec une carte TNT ou Radeon, les manipulations suivantes peuvent améliorer les performances :

 

 

Effectuez une copie de sauvegarde du fichier DeusEx.ini situé dans le répertoire " System " de votre jeu. Cette copie vous permettra de restaurer votre configuration en cas de problème.

 

Affichez le fichier DeusEx.ini à l'aide d'un éditeur de texte.

 

Dans le fichier, recherchez la portion de texte commençant par " [OpenGLDrv.OpenGLRenderDevice] ".

 

Dans cette même section, vous devriez trouver, quelques lignes plus bas, la chaîne de caractères " UseTNT=0 ". Modifiez-la en " UseTNT=1 ".

 

 

(Attention : en mode OpenGL, la capture du menu de chargement ne sera pas affichée dans les parties sauvegardées. Cela est normal.)
