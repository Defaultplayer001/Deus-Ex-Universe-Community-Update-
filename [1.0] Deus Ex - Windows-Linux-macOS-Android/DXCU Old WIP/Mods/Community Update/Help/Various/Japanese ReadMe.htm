４) ディスプレイ設定

『Deus Ex』では、最高のパフォーマンスを実現できるよう、多くのグラフィック オプションを変更できます。システムでゲーム速度が遅くなった場合は、ゲーム オプションとディスプレイ オプションを変更させることで、ゲームのパフォーマンスを最適化できます。ディスプレイの設定は、[Setting](セッティング)を選択して、[Display](画面設定)をクリックすると表示されます。

以下にリストした設定では、パフォーマンスを大きく変更できます。特に、ワールド テクスチャ ディテール、画面解像度、テクスチャの色深度が有効です。ゲーム オプションとディスプレイ設定の完全なリストについては、ゲームのマニュアルを参照してください。

[Adjust Brightness]

（輝度調整）

　ゲーム画面が暗過ぎる、または明る過ぎる場合、このオプションを選択してスライダーを動かすことで、
　全体の明るさを変更することができます。
[Toggle Full-Screen Mode]

（フルスクリーン モード切り換え）

　この設定でウィンドウとフルスクリーンのどちらかに切り換えます。
　『Deus Ex』では、常にフルスクリーン モードでプレイすることをお勧めいたします。
[Rendering Device]

（レンダリング デバイス）

　このオプションを選択すると、3Dグラフィックス モードを変更できます。
　ゲームを終了して再起動すると、[Video Configration]（ビデオ設定）画面が表示されます
　（このファイルの「ビデオ設定」セクションを参照してください）。
　レンダリング デバイスを変更する前にゲームを必ず保存してください。
　保存していないと、現在のゲームが失われてしまいます。
　公式にサポートしているモードは、3Dfx Glide for WindowsとDirect3D Supportだけです。
 他のモードを使用している場合は、サポートしているどちらかのモードに変更することを強くお勧めします。
[Screen Resolution]

（画面の解像度）

　ゲームで画面の解像度を低くすると、フレーム レートが改善されます。
　現在のシステムでゲーム速度が遅い場合は、画面の解像度を低く設定してください。
[Texture Color Depth]

（テクスチャの色深度）

　ゲームは、16ビット カラー モードで最も快適にプレイすることができます。
　すべての3Dカードが、32ビット カラーをサポートしているわけではありません。
　テクスチャの色深度での変更を有効にするには、ゲームを1度終了してデスクトップに戻り、
　ゲームを再起動してください。

　注意: 
　1度にテクスチャの色深度と画面の解像度を変更することはできません。
　画面の解像度をまず設定して[OK]をクリックし、次にテクスチャの色深度を変更するために、
　ディスプレイ設定画面に戻ってください。
[World Texture Detail/
Object Texture Detail]

（ワールド テクスチャのディテール/
オブジェクト テクスチャのディテール）

　ディテール レベルを低く設定すると、カードのパフォーマンスが改善されます。
　ただしこれを行うと、テクスチャの鮮明さが低くなります。

『Deus Ex』では、最新の優れたサウンド カードでサポートしている以下のようなオプションが利用できます。

音とびや音の途切れなど、サウンドの問題が発生する場合は、サウンドカード ドライバを更新してください。
また、スモール インストールの設定でゲームをプレイしている場合は、ゲーム コンポーネントをすべて選択して、インストールをやり直してください。

[Effect Channels]

（エフェクト チャンネル） 

 　エフェクト チャンネルの数を減らすと、パフォーマンスが改善されます。
[Dolby Surround Sound]

（ドルビー サラウンド サウンド）

　ドルビー サラウンド サウンドを使用するには、コンピュータにステレオ サウンド カードを装着して、
　Dolby Pro Logic Surround Sound対応スピーカーを接続しなければなりません。 
[Hardware 3d Support]

（ハードウェア3Dサポート）


　サウンド カードで3Dサウンドや環境サウンドをサポートしている場合（EAX、A3Dなどに準拠したサウンド カード）、
　ハードウェア3Dのサポートを利用すると非常にリアルなサウンドを体験できます。
　ただし、サウンド カードがハードウェア アクセラレーション、3Dサウンド、環境サウンドをサポートしていない場合、
　ハードウェア3Dのサポートをオンに設定すると、サウンドが何も出力されなくなる可能性があります。
　この現象が発生した場合は、オーディオ オプションに戻って、ハードウェア3Dのサポートを[OFF]に
　設定しなおしてください。

　注意：
　 このオプションを変更した場合、変更を有効にするには、ゲームを終了してデスクトップに戻り、
　再起動してください。

　

　
６）セーフモードについて 

『Deus Ex』がお使いのPCで正常に起動しない場合、問題解決の為セーフモードを利用することをお勧めします。
セーフモードで以下のオプションを外すことにより、問題のある項目を回避することをお試しください。

以下の２つの方法でセーフモードが立ち上がります。

①ゲームプレイ中の強制終了後のゲーム起動時　（この場合は自動的にセーフモードが起動します）

②任意にセーフモードを立ち上げた場合

　

◆セーフモードの起動方法

１．WindowsMe/98/95を起動させ、デスクトップ画面を表示させます。
２．タスク バーの [ｽﾀｰﾄ] → [ﾌﾟﾛｸﾞﾗﾑ] → [デウスエクス] → [デウスエクスセーフモード] を選択してください。
３．その後プログラムの指示に従いセーフモードを実行してください。

　

◆『Deus Ex』をセーフモードで起動する場合、以下のオプションが選択できます。
Disable all sound ゲーム中のサウンドを起動しません。

注意：このオプションを選択してゲームを起動した場合、ゲーム中のサウンド設定ができなくなります。
Disable 3D sound hardware ハードウエア側の3D サウンドの機能を使用することができなくなります。
Disable 3D video hardware ハードウエア側の3D グラフィックの機能を無効にします。
Run the game in a window　　　　　　　　　　　　 (rather than fullscreen) ウィンドウ モードでゲームを起動します。
Run in standard 640x480 resolution 標準の解像度　640×480　でゲームを起動します。
Reset all configuration options to defaults ディスプレイとサウンドの設定およびオプションのGore Levelをデフォルトに戻します。
Disable Pentium III/3DNow processor extensions Pentium IIIおよびAMD Athlon（3DNow!）の拡張機能を使用することができなくなります。
Disable joystick support ジョイスティック・サポートを使用する事ができなくなります。

　

７）その他のパフォーマンスに関するヒント　
『Deus Ex』は、64 MBのRAMでもご利用いただけますが、スムーズなプレイをお楽しみいただくには、少なくとも128 MBは必要になります。実装されているメモリの容量に関わらず、以下の作業を行うことによって、利用できるメモリ容量を増やし、ゲーム エンジンの全体的なパフォーマンスを改善することができます。


開いているウィンドウを閉じる。→　ICQのようなメニュー バー プログラムを含めて、他のすべてのプログラムを終了する。→　ハード ドライブを最適化する。　→　ハード ドライブに空き容量があるか確認する。


ゲームのインストール時に、全ゲーム コンポーネントをインストールするフル インストールを実行すると、ゲームはよりスムーズにプレイできるようになります。インストール後にインストール タイプを変更したい場合は、『Deus Ex』をアンインストールして、ゲームのインストールを再実行してください（ハード ドライブに十分な空き容量があるか確認してください）。この場合でも、保存したゲームは削除されません。

８）ゲームの保存/ロード

ゲームのプレイ中は、いくつかのマップを順番に進んでいき、1つのマップを終了したら、次のマップに移ります。
これらのマップを進む場合、次のマップに移動するときに、[SAVING](保存中)、[LOADING](ロード中)という文字が画面に順番に表示されます。
ゲームでは前のマップの状態が自動的に保存されますが、ゲーム自体は自動的には「保存されません」。
したがって、ゲーム メニューでゲームを手動で保存するか、あるいはホットキーのクイック・セーブを押す必要があります。
９）ゲームプレイについて


a)ハシゴの昇り降り

高い場所からハシゴを下ろす場合、あるいは、ハシゴで下に降りる場合、[Crouch]（しゃがむ）ボタンを押すと、高い場所からハシゴを下ろす（そして、下に降りる）ときにキャラクターが慎重な姿勢をとることによって、ハシゴから「落ちる」という危険性を少なくできます。このように、ハシゴの昇り降りのときに[Crouch]ボタンを押せば、キャラクターがしゃがみ、岩棚などでの移動が簡単になります。

ハシゴの昇り降りの際には、多少、左右や前後に身体を動かすことができます。ただし、ハシゴから外れて下に落ちないように十分に注意してください。落ちないようにするには、つねにハシゴに注意を払い、ハシゴの中心線から外れないようにすることです。特にハシゴで降りる場合は注意が必要です。下に降りるときに、よそ見などをして注意を怠っていると、中心線からずれて下に落ちてしまうことがあります。
b)十字線


ゲームプレイ中、画面の中心にある十字線を人や物に合わせると、合わせた人物または物体の性質によって色が変わります。

十字線が緑である場合は、標的が友好的または中立的な人物や動物であることが分かります。
十字線が赤である場合は、標的が敵であることが分かります。
十字線が白である場合は、標的が物体であることが分かります。

敵に見えるようなキャラクターでも、実際は友好的な場合があります。
敵かどうか確認せずに味方を攻撃してしまうと、周囲の味方である人物も攻撃をしてきますので、十字線の色には十分な注意を払ってください。
c)セキュリティ・システム


セキュリティ ・システムへのログインまたはハッキングが成功した場合、セキュリティ ・システムの画面が表示されます。この画面では、カメラ、ドア、ロックを操作できますが、上位のスキル レベルがあれば、以下のように自動迎撃システムまで操作できます。

Turret Statusが[味方]に設定されていると、自動迎撃システムは、システムをセットした味方以外を攻撃します。 
Turret Statusを[敵]に設定すると、自動迎撃システムは、システムをセットした味方を敵と認識し攻撃します。 
Turret Statusを[すべて]に設定すると、自動迎撃システムは周辺に存在するあらゆる人物と物体を攻撃します。
Turret Statusを[バイパス回路接続]に設定すると、自動迎撃システムは完全に使用不能になります。
d)腕への攻撃

Arm（腕）を攻撃すると、敵が武器を握られないようにできます。
武器が地面に落ちるまで、敵のArmを繰り返し攻撃してみましょう。
e)スキル レベル

スキル レベルを軽視してはいけません。『Deus Ex』のゲーム プレイを進めていると、このレベルの違いが大きな差を生みます。

ゲームのプレイ中は、スキル ポイントを獲得して、スキル レベルを上げることを忘れないでください。 
Weapons（武器）以外のスキルは、ゲームを進めていく上でWeaponsのスキルに勝るとも劣らずに強力なものです。

Lockpick（ロックピック）やMultitool（マルチツール）が頻繁に足りなくなるような場合は、Lockpicking（ピッキング）やElectronics（エレクトロニクス）のスキルを「Trained」(訓練済み)またはそれ以上に高くすると良いでしょう。

Computer（コンピュータ）、ATM、Security System（セキュリティ ・システム）などに侵入したい場合は、Computer（コンピュータ）のスキルを「Trained」(訓練済み)またはそれ以上に高くする必要があります。

ポイント: すべてのスキルでMasterになることは非常に困難です。したがって、スキルポイントを割り振る場合には注意が必要です。1度ポイントを使ってしまうと、これらを取り戻すことはできません。

Augmentation（能力増幅）
Augmentationは、自分の意志でオンとオフを切り換えられる特殊な能力です。Augmentationを使えるようにする為にはAugmentation Canister（能力増幅キャニスター）が必要で、またその能力を高めるにはAugmentation Upgrade Canister（アップグレード キャニスター）が必要になります。通常、これらのアイテムは発見が難しい場所にあります。しかし、これらのキャニスターの価値は非常に高いため、探すには十分な時間をかけて必要な手間を惜しまないことが重要です。

Combat & Stealth（戦闘と隠密行動）
ミッションの使命を達成するために戦うことはよくあることですが、戦っていくことだけが「最善」の選択肢であるとは限りません。冷静な状況分析と知恵によって、敵の裏をかくことも必要になります。音を立てない、敵やカメラの視界に入らない、周りの物などを巧みに利用するといったことに注意してください。
f)行き止まり

周りをよく確かめ、周囲の人に声をかけ、床や壁に通気口やマンホールなどがないか探し、巧妙に隠されたハシゴ、ドア、キー・パッドを見つけ出してください。
目的の場所にたどり着くには、いくつかのルートが見つかるでしょう。

１１）ゲームのヒント

●Looking Around
　（周囲を見回す）		 ゲームが始まってまず実行すべきことは、周囲を注意深く観察することです。J.C.の視線はマウスを使ってコントロールできます。

■十字線：視線の中心には十字線が表示されます。この十字線を基準にすると、戦闘の際に狙いを定めたり、アイテムを拾って使ったりすることが簡単になります。ゲーム オプションを使えば、この十字線を消すこともできます。
●Focusing
　（フォーカス）
 操作可能で手が届く範囲にある対象物（人物も含まれます）に十字線が重なると、自動的にそこに焦点が合います。すると、その名前が表示されます。

フォーカスしているときに対象物を右クリックすると、Dentonは最も適切な方法でその対象物を「使用」します（小さなアイテムを拾う、ドアを開ける、本を読むなど）。
●Vision Enhancements
　（視野の拡大）		 このゲームには視野を広げるアイテムがいくつか登場します。

＜D.Clouds Military Binoculars＞（双眼鏡）を使用すると、遠くのものが大きく見えます。

＜Telescopic Sight＞は、双眼鏡と基本的に同じですが、使用方法が異なります。Telescopic Sightを通して何かを見たいときは、これを装着した武器（PIS LRW350 Sniper Rifleなど）か、スコープ　アップグレードした物を手に持ち、[ [ ] キーを押してください。ただし、武器のスキルが低かったり、標的が遠すぎたりすると、視界が揺れる場合があります。

＜Light Amplification Goggles (LAG)＞（暗視装置）も双眼鏡に似ていますが、わずかな光を増幅する技術が用いられている点で異なります。少しでも光がある環境なら可視能力が高まります。ただし、LAGは使い捨てで、使用時間にも制限があります。
●テキストとデータ表示 このゲームに登場するアイテムには、テキストや数字の形で情報をもたらしてくれるものが数多くあります。＜Book＞（本）、＜Newspaper＞（新聞）、＜Data Cube＞（データキューブ）などはその代表です。普通、これらのアイテムを拾ってアイテム リストに加えることはできません。そのかわり、右クリックすることで、データが記されたテキスト ウィンドウが表示されます。アイテムから視線をずらすと、このテキスト ウィンドウは閉じてしまいます。ミッションに重要なデータであった場合は、メモとして[Goals/Notes]画面にセーブされます。また、イメージの場合は、イメージ画面にセーブできます。

対話型のデータ表示：＜Email＞、＜Security Systems＞（セキュリティ・システム）などは、対話形式でデータ表示が行われます。コードまたはコマンドの形式で、情報を入力してください。データ表示画面には入力方法が表示されます。
●キャラクターの動き デフォルト設定の状態では、矢印キーがキャラクターを動かすための基本ツールとして使われます。

J.C.は顔を向けている方向に前進します。そのため、角を曲がりたいときは、前進しながら右または左に顔を向ける必要があります。

平行移動：体の向きを変えずに、右または左に進むことを平行移動と言います。

前かがみ：右または左の方向に前かがみになると、曲がり角の先をちらりと見ることができます。前かがみの姿勢は、キーを押している間だけ保たれます。

速度：基本的な移動方法には「歩行」と「走行」の2つがあります（Augmentationを使用する事でさらに速く走れるようになります）。初期設定としての移動は、走行です（これはオプション メニューで変更できます）。ただし、歩行やしゃがんだ状態で移動する方が物音を立てなくてすむため、その分、敵に見つかりにくくなります。

ジャンプ：裂け目を飛び越えたり、アイテムの上に跳び乗ることができます。ジャンプの飛距離は助走の速度で決まります（Augmentationのレベルやダメージの状態で変動します）。

■ハシゴ：ゲーム全般を通じて、ハシゴを昇り降りする機会が数多くあります。ハシゴを昇りたいときは、ハシゴの中心線にJ.C.の身体を合わせ、矢印キーを使って普通と同じように移動します。ハシゴの正面にいるのに昇り降りができない場合は、試しにジャンプしてみてください。ハシゴを降りるときは、あらかじめしゃがんでおくと、ハシゴから落ちる危険性が少なくなります。

■Swimming（水泳）：泳ぐ場合も、使用するキーはほぼ同じです。ただし、前後左右に加えて、上下方向の移動が可能になります。泳いでいるときにマウスを動かして顔を下に向けると潜ることができ、逆に、顔を上げると浮き上がることができます。あまり長い間潜っていると、酸素不足でダメージを受け、最終的には溺死してしまうことがありますので注意しましょう。
●Stealth
　（隠密行動）		 ほふく前進、つまりしゃがんだままの状態で移動することもできますが、その場合は移動速度が遅くなります。

この「しゃがむ」という動きは、このゲームにおいて重要な意味を持ってきます。まず、あまり高さのないアイテムの後ろにでも身を隠すことができます。また、入口が小さくても入っていくことができます。

さらに重要なのは、しゃがんでいると敵に見つかりにくくなるという点です。遮蔽物を最大限に利用できるようになり、ほとんど音を立てずに移動します。

当然ながら、このような隠密活動にも限界があります。あらゆる場所で電子センサーが作動していますし、開けた場所や敵がすぐ近くにいる場合には、見つかる可能性が高くなります。少しでも音を立てれば、たちまち見つかってしまうでしょう。
たとえしゃがんでいても、誰かを攻撃すれば、おそらくは周囲にいる者に気付かれてしまうでしょう（もちろん、スキルや巧みな戦術を身に付ければ、敵を静かに素早く倒すことは可能です）。

しゃがんだ状態でのジャンプ：背の低い障害物の上をほふく前進したり、小さな窓を通り抜けたりしなければならないときは、しゃがんだ状態でジャンプすることも可能です。
●装備の使用 ミッション中は、拾って持ち運べるアイテムが見つかります。Inventoryに加えて持ち運べる物は、何かの役に立つはずです。

手に持つ最も簡単な方法は、画面の下に表示されている9つのアイテムベルトにアイテムを入れておくことです。これらのボックスには番号が付けられており、対応する数字キーを押すとそのボックスのアイテムを手に取ることができます。

備蓄可能なアイテム：このようなアイテムには、＜Grenade＞（グレネード）、＜Food＞（食料）、＜Medkit＞、＜Bioelectric Cell＞(バイオエレクトリック・セル)、＜Lockpick＞（ロックピック）、＜Multitool＞（マルチ ツール）などがあります。

1つのボックスには、1種類の備蓄可能アイテムを蓄えられます。ただし、ボックスが満杯になると、貯まっているアイテムを減らさない限り、それ以上は同じ種類のアイテムを備蓄できなくなります。

■複数アイテム：備蓄可能なアイテムは、個別のボックスに入れることでいくつも持ち歩くことができます。その他のアイテム（武器の大部分）については、Inventoryに1つしか入れられません。つまり、すでに1つ持っている場合には、2つ目を見つけたとしても所持できないのです（ただし、そうすることが可能である場合に限り、2つ目の武器から弾薬をInventoryに移し替えることができます）。

■アイテムのサイズ：ある種のアイテム、とりわけ大型の武器は、Inventoryの中で複数のボックスを占めます。Rocket（ロケット）系、Rifle（ライフル）系、Sword（刀）系などは特にかさばります。

なお、Inventoryのボックスを数多く占める武器であっても、アイテムベルトのボックスは1つしか必要としません。
●Nanokeys
　（ナノキー）		 Nanokeyとは、ナノ・マシンの固有パターンそのもので、特定のナノロックを開くために使われます。これは小さな青い円筒に詰められた形で発見され、Nanokey Ringの中に保管されます。Nanokey Ringには、Inventoryのボックスは必要ありません。Nanokey Ringは常にアイテムベルトの最後のボックス（0）に表示されており、この表示を消すことはできません。

Nanokey Ringは手に持つことが可能で、ロックされているドアに関して一般的なやり方で使用します。キー・リングにナノロックに関して適切な情報を持つナノキーがあれば、ロックを解除することができます。
●アイテム 防御や救急用のアイテム（＜Ballistic Absorb Armor＞（バリスティック・アブソーブ・アーマー）、＜Light Amplification Goggles(LAG)＞、＜Rebreather "Blue Mermaid"＞（レブレザー・ブルーマーメイド）など。ただし、これらに限られません）には、エネルギーを内蔵しているものがあります。いったん起動すると、このタイプのアイテムは、エネルギーが尽きるまで稼動しつづけます。再使用したり、Inventoryに戻したりすることはできません。通常、アイテムのエネルギーは数秒、あるいは数分で使い果たされます。
●Inventoryの管理 ミッションを通じてアイテムを保管しておける場所は、Inventoryのみです。それ以外の場所にアイテムを残しておいても、ミッションが変わればそのアイテムも消えてしまいます。残しておいた場所に行ったとしても、アイテムは見つからないでしょう。

それだけに、最も役に立ちそうなアイテムだけを所持することが極めて重要となります。なお、ゲームに不可欠なアイテムに関しては、手離すことはできません。
●武器のアップグレード 弾薬の再装填が可能な武器は、いくつかの方法でアップグレードできます。これらのアップグレードは、ゲーム中にMODとして見つかります。武器をアップグレードする際は、Inventoryでアップグレードしたい武器の上にMODをドラッグします。いったん武器をアップグレードしてしまうと、このアップグレードを取り消したり、他の武器に適用したりすることはできなくなります。アップグレードした武器を捨てると、アップグレードも全て失ってしまうことになります。
アップグレードできる武器を確認したいときは、Inventoryをクリックしてください。全てのアップグレード可能な武器が反転表示されます。

１つの武器に1回しか使えないアップグレードもいくつかあります（＜Scope＞（照準器）、＜Silencer＞（サイレンサー）など）。その他のアップグレード（＜Accuracy＞（命中精度向上）、＜Range＞（射程距離延長）、＜Recoil＞（反動制御）、＜Clip＞（弾倉拡張）などの改良）は何度も使えますし、組み合わせて効果をいっそう高めることもできます（ただし、同じようなアップグレードを組み合わせると、次第に効果が薄くなっていきます）。

武器によっては、最初から「標準的な」アップグレードが施されています。たとえば、＜PIS LRW350 Sniper Rifle＞（PIS LRW350　スナイパー ライフル）の場合、＜Telescopic Sight＞が初めから装着されています。＜PIS MSK4600-B Stealth Type＞（PIS MSK4600-Bステルスタイプ）、＜Mini-Crossbow "Night Sleeper "＞（クロスボウ “ナイトスリーパー”）などに関しては、SilencerのMODを使う必要がありません。
●Communicating
　（コミュニケーション）		 友好的な人物（IFFで「敵」と判断されなかった者）をクリックすると、大抵の場合は何かを話してくれます。急いでいない（たとえば、銃などで狙われていない）場合は、できるだけ多くの人に声をかけた方がよいでしょう。

■会話：ゲーム中には、とりわけ重要な意味を持つキャラクターがたびたび現れます。こういったキャラクターに近づいて、それらをクリックすると、画面が「第三者」の視点（カットシーン）に変わって2人の会話が始まります。

いったん始まった会話を途中で止めることはできません。会話中に攻撃されることはありませんが、話している最中に敵が接近してきて、会話が終り次第攻撃を開始しなければならないような可能性は十分にあります。

■インフォリンク メッセージ：インフォリンクを通じて、本部からときおり連絡が入ります。インフォリンク メッセージは、重要な警告または連絡事項です。インフォリンク メッセージを受信しても、J.C.の行動が邪魔されることはありません。

■記録：現在遂行中のミッションにおいてJ.C.が耳にしたこと（独白、会話、インフォリンク メッセージ）は、すべて [Conversations]画面にセーブされます。ミッション遂行のために欠かせない情報（パス コードなど）が会話に含まれていた場合、その情報は、[Goals/Notes]画面に永続的に記録されます。
●戦闘 敵を攻撃するには、武器を手に持っていなければなりません。武器を手にしたら、射程内に入っていることを確かめた上でターゲットにカーソルを重ね、クリックします。

注意：早まって武器を手にしたまま進んでいくと、本来なら友好的な場所の多くが危険な敵地に変わってしまいます。
●照準 「Deus Ex」では、誰を撃っているかだけではなく、どこを撃っているのかを知ることも重要です。以下のように、プレイヤーと同じく、敵も身体の特定部分にダメージを受けます。

Head（頭）を撃つと、致命的なダメージを与えられますが、奇襲や至近距離からの攻撃でない限り、命中させるのは至難の技です。

Torso（胴体）は命中しやすく、最終的には敵を倒すことができますが、そのためには何発も撃ちこむ必要があります。

Arm（腕）とLeg（脚）は命中しやすく、敵の動きを封じることもできます。しかし、致命傷にはなりえません。

■照準器：武器を構えると、照準（十字線）が表示されます。また、＜Telescopic Sight＞を使おうとすると、照準が揺れることがありますが、スキルを高め、武器をアップグレードすれば、銃を持つ手が安定して、狙いをつけやすくなります。

ターゲットを捉えた直後の照準は広がった状態になっています。しかし、狙いつづけると照準は徐々に狭まっていきます。完全に狭まった瞬間に命中率は最大になります。移動すると照準はリセットされてしまいます。

2つの特殊照準器はオンとオフの切り換えに特別なコマンドを必要とします。

＜Telescopic Sight＞（スコープ）は [ [ ] を使って切り換えます。

＜laser Sight＞（レーザーサイト）は [ ] ] を使って切り換えます。なお、レーザーサイトを使うと小さな光点が生じるので、敵が注意深いとこちらの存在を悟られてしまうことがあります。
●武器のタイプ 「Deus Ex」には、何種類もの武器が登場します。鈍器、ナイフ、刀剣、ピストル、ライフル、突撃銃、各種重火器はもちろんのこと、小型のクロスボウや投げナイフなど、特殊な武器も用意されています。

以下は様々な武器に関する一般的なアドバイスです。

■接近戦：言うまでもなく、重要なのは武器の有効射程内に敵を入れることです。
接近戦用の武器の大半（＜Crowbar＞（バール）、＜Baton＞（警棒）、＜Neomaxter Combat Knife＞（Neomaxterコンバットナイフ）、＜Sey-ryu- Sword＞(青龍刀)などは弾薬を必要とはしません。しかし、中にはエネルギーを必要とするものがあります（＜SIS RSW20 Riot Prod＞（SIS RSW20ライオット プロッド）など）。

■非殺傷用の武器：＜SIS RSW20 Riot Prod＞のような武器だけが、敵を殺傷することなくその動きを封じることができます。ほんの一時的に敵を気絶させることしかできない場合もあれば、ある程度の時間気絶させておくことができる場合もあります。一般的に、気絶しても倒れることなくそのまま立っているような敵は、すぐに逃げ出そうとするか、さもなければ闇雲に襲いかかってきます。

■突撃銃：＜PIS XMS214 Assault Rifle＞（PIS XMS214アサルト・ライフル）のようなオートマチックの武器は大きな殺傷力を秘めていますが、反動によって銃口が上に向こうとするので、高いスキルと武器のアップグレードが、この問題の改善に効果があります。

■使い捨ての武器：＜PIS Light Anti-Tank Weapon(LAW)＞（LAW＝携帯用対戦車兵器）に代表されるような武器は、使用が1回に限られています。使ってしまうと、このタイプの武器はアイテム リストから消えてなくなります（ただし、別のLAWを後で見つける可能性はあります）。

■広域破壊兵器：とりわけ強力な武器、たとえば＜Guided Explosive Projectile (GEP) Gun＞（ＧＥＰガン）や＜LAM＞、＜Grenade＞（手榴弾）、＜Flamethrower CT-01＞（火炎放射器）などは、広い範囲に効果を及ぼします。したがって、このタイプの武器は、敵が集団で襲いかかってきたときに効果的です。しかし、敵との距離が不十分だと、自分の武器で負傷したり命を落としたりする危険性があるので注意してください。
＜Grenade＞の扱いは特に注意を要します。
●Reloading & Ammunition
　（再装填と弾薬）		 弾切れをおこした武器への再装填は自動的に行われます（弾薬があることが前提条件）。[Game Option]画面を表示すれば、自動装填をオフに設定することもできます。
戦闘中の再装填は、貴重な時間の損失となります。＜Clip＞に十分な数の弾が残っていることを確認しておきましょう。確認するには、手に武器を持った状態で [:] キーを押してください。

弾薬はInventoryの場所を取りません。Inventoryで武器をクリックすると、その武器に装填可能な弾薬が全て表示され、現在所持しているものは明るい色で示されます（まだ獲得していない武器の弾薬を持ち歩くことも可能）。

特定の種類の弾薬をどれだけ持っているのかを確認したいときは、[Inventory] 画面で [弾薬] ボタンをクリックします。すると、手持ちの弾薬がリストアップされ、所持している武器に必要な種類が示されます。

[Inventory] 画面で武器をクリックして、装填したい弾薬をクリックすると、弾薬の種類を変更できます。またキーボードによって違いますが、武器を手に持って [#] または ['] キーを使っても、弾薬を変更することができます。
●ブービー ・ トラップ ＜LAM＞と＜Grenade＞には、粘着剤と接近感知センサーが付けられています。これらは即席の地雷やブービー・トラップとして利用できます。

ブービー ・トラップを仕掛けたいときは、手榴弾を用意して、手近にある平らな面（壁など）をクリックしてください。数秒後に接近感知センサーが起動して、手榴弾のセンサー圏内に何者かが入ると爆発するようになります（自分が仕掛けたブービー･トラップが、自分を感知して爆発することはありませんが、近くの敵によって爆発した場合は、自分も巻き込まれる可能性があります）

敵もブービー・トラップを仕掛けることができるので要注意です。手榴弾に素早く近づき、爆発する前に右クリックすれば、トラップを無効にできます。無効にした手榴弾をもう1度右クリックすれば、拾って自分のものにすることができます。
●ダメージと回復 ■負傷
負傷したときに重要なのは、身体のどの部分にダメージを受けたのか覚えておくことです。

■Healing（治療）
「Deus Ex」に「自然治癒」という言葉はありません。傷を治したいときは、何らかの治療薬を手に入れなければなりません。主な治療方法には、以下のようなものがあります。

体力画面から使用するMedkitは、身体の特定の部分に対して用いることができます。その他の治療薬は、J.C.の体内システムが最も必要と判断した部分に対して用いられます。

▲Medkit：このコンパクトな万能薬は、肉体的なダメージを瞬時に回復させてくれます。治療する部分やスキルによって変わりますが、Medkitは30ポイントから50ポイントのダメージを治してくれます。

▲MedBot：これは半自動型の移動式手術ユニットです。MedBotを使用すると、最大300ポイントのダメージから回復できます。同じMedBotを繰り返し使うことができますが、1度使用すると、1分間の充電時間が必要になります。MedBotはAugmentationの組み込みにも必要です。

■Augmentation：ある種のNano-Augmentationは、ダメージからの回復を可能にします。

■食料：J.C.の体内のナノシステムは、合成炭水化物を摂取して、それを肉体的なダメージを回復させる「燃料」に変える能力を備えています。食料は、Inventoryとアイテムベルトのどちらからでも摂取できます。1個の＜Chocolate Bar＞（チョコレート）では回復できなくても、3個の＜Chocolate Bar＞（チョコレート）と4本の＜Soda＞（ソーダ）、それに1個の＜Soy Food＞（大豆フード）を食べれば、かなりの治療効果を発揮します。食料による治療を、身体の特定の部分に割り当てることはできません。

アルコール飲料（＜Forty＞（ビール）、＜Wine＞（ワイン）、＜Liquor＞（蒸留酒））にも多少の治療効果があります。ただし、アルコールを摂った後は、しばらく頭の回転が鈍くなるので注意してください。タバコには全く治療効果がありません。
●Bioelectric Energy
　（バイオエレクトリック・エネルギー）
能力増幅は、生体電気エネルギー、いわゆるBioelectric Energy(バイオエレクトリック・エネルギー)で機能します。身体がバイオエレクトリック・エネルギーを自然に生み出すことはなく、体内の「電池」にこれを蓄えなければなりません。このエネルギーを使い果たすと、Augmentationを使うことができなくなります（＜Infolink＞（インフォリンク）、＜IFF＞は例外です。これらはバイオエレクトリック・エネルギーを使用しません）。

ダメージ データ表示の左側にバーがあり、この高さで現在のバイオエレクトリック・エネルギーのレベルを確認できます（エネルギーが多いとバーが高くなります）。Augmentation画面のグラフで確認することもできます。

少なくなったバイオエレクトリック・エネルギーは、以下の2つの方法で補充できます。

■Repair Bot（修復ロボット）：Bioelectric Energyを補充してくれる（最大容量の75%まで）便利な小型マシーンです。Repair Botは何度でも使えますが、1度使用すると、1分間の充電時間が必要になります。

■Bioelectric Cell(バイオエレクトリック・セル)：備蓄が可能です。Bioelectric Cellは、よく使用することになることでしょう。このアイテムは使い捨てです。エネルギーを使い果たした状態から完全に補充するのには、4つのBioelectric Cellが必要です。
●Lockpicking
　（ピッキング）		 Lockpickは先端にキーロック解除に必要な正しい鍵の形状に変化する凝固性ポリマーを備えた、最新の開錠アイテムです。Lockpickは使い捨てのアイテムですが、いくつものLockpickを使わないと正確なキーパターンを作り出せない場合もあります。

ロックされているドアに近づくと、J.C.の体内のセンサーがそのロックを分析して、侵入するのに必要なLockpickの数を予測します。この情報は自動的に表示されます。ドアを開くことができない場合は、ロック強度に「Locked」と表示されます。

■ドアの破壊：センサーがドアの物理的な強度も推定してくれるので、物理的にドアを破壊できるかどうかが分かります。ドアは爆発物や重火器を使うことで破壊できます。ドアを破壊できない場合は、ドア強度に「INF」と表示されます。
●Electrical Instruction
　（ハッキング）		 このやり方は、Lockpickとほとんど同じですが、暗証キーによるロックやセキュリティ カメラ、それに自動迎撃システムを無効化できるという点が異なっています。ハッキングにMultitoolを使用する場合があります。Lockpickと同様に、内蔵センサーがセキュリティ・システムを分析して、無効化するのに必要なMultitoolの数を予測します。

暗証キーを備えたドアの中には、＜Multitool＞でしか開けられないものがあります。そうでないドアは、MultitoolとLockpickのいずれでも開けられるため、状況に応じてより効果的な方を選んでください。

　

１２）ゲームプレイに関するご注意 


◆光過敏性てんかんに関するご注意
ゲームをプレイされるときは、必ず次のことに注意してください。
特定の光の点滅を見たり、日常的に存在する光の明減パターンを見ることによって、てんかんの発作を起こしたり、意識を失う体質の方がいらっしゃいます。そのような方は、ゲームやアニメーション/ムービーの画面を見ることで発作が引き起こされる心配があります。こうした症状は、てんかん、またはてんかんに近い症状（発作または失神）を経験されていない方にも起こる可能性があるため、ゲームをプレイされる前に、医師の検査をお受けになることをお薦めします。また、とくに小さな子供たちだけでゲームをプレイする場合は、子供たちの様子に変化がないかどうか、周囲の大人がつねに気をつけて見ていることが大切です。
プレイ中に次のような症状が現れたときは、即座にゲームを中止して、医師の診察を受けてください。
めまい、目のかすみ、目または筋肉のけいれん、失神、方向感覚の喪失、身体のけいれん、ひきつけなど。

◆ゲームプレイに関する注意点
●画面に近づきすぎないようにしましょう。コントローラのケーブルがいっぱいに延びるくらいの距離まで画面から離れて座りましょう。
●小さめのコンピュータディスプレイでプレイされることをお勧めします。
●疲れているとき、また寝不足のときはプレイしないようにしましょう。
●プレイする部屋を明るくしましょう。
●１時間プレイしたら１０分から１５分の休憩をとるようにしましょう。
